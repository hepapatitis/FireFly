C) CSS Files and Structure - top

Styles

There is only one main CSS file in this theme. Please use custom.css to override the existing CSS file.

Lets take a look closely at our stylesheet files.

bootstrap.min.css (Bootstrap ONLY)

Base styles for Bootstrap 3.2.0.

font-awesome.min.css

Base styles for FontAwesome icon fonts.

style.css

Main CSS that includes Reset CSS by Meyer.

YTPlayer.css (Video Background ONLY)

CSS file for YTPlayer that is only used for Video Background style. Be sure to include this file when using Video Background.

custom.css

Please put all your custom changes to this file. This CSS has to be loaded last to override perfectly.

F) FAQs - top

How to work with contact form?

This template contains fully working contact form. All you need is replace you actual e-mail address. Open the send-email.php file that is located in the root with any text editor and replace all instances of "your-email@yourdomain.com" with your email address in the entire file using the Find and Replace tool (usually the Ctrl+H hot key).

This coming soon page is laggy, why is that?

Parallax and FireFly are intensive plugins that can take a lot of memory resources, especially screen with big resolution. If it's too much, you can remove FireFly js to make it less laggy.

What should I do to use Youtube video as background?

First of all, make sure that you include YTPlayer.css and jquery.mb.YTPlayer.js into index.html. Those files are not included in other styles such as layered-parallax, single-background, and multiple-background.

After that insert the following HTML tag in index.html.

	My video

Your index.html should be like this:

	...
	
	
	
	
	...
	
	My video
	
	...
	
	
	
	
	...

To change the youtube video, simply change the link in videoURL to any YouTube URL that you want.

Now open js/main.js and make sure to include this line:

	jQuery("#bgndVideo").mb_YTPlayer();

And voila! It's done.

The video background is not displaying anything. What should I do?

Video Background will work when it's opened from a webserver (in this case, your webhosting), not your local machine. You can use XAMPP to create a local webserver.

How do I change FireFly images or total number of FireFly?

FireFly uses PNG image which can be found in js/main.js. Below is the sneak peek of main.js for FireFly.

/*----------------------------------------------------------------------------*/
/*	7.	Firefly
/*----------------------------------------------------------------------------*/	
	$.firefly({
		images : ['images/in-the-depths/fly1by1.png', 'images/in-the-depths/fly2by2.png', 'images/in-the-depths/fly3by3.png'],
		total : 30,
		on: '.parallax-bg'
	});

To change the FireFly picture, simply change the images with the image that you desire.
To change the total number of FireFly, simply change the total value.
